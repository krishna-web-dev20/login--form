## Login Form
A simple and responsive Login Form project built using HTML, CSS and JavaScript. Validation and a clean modern user interface.
##Live Demo
https://krishna-web-dev20.github.io/login--form/
## Features
-> Input Validation 
-> Email Format Checking
-> Password Validation
-> Local Storage Support
-> Fully Responsive Design 
-> Smooth User Experience
-> Clean UI Design
-> Show/Hide Password Feature
-> Empty Field Warning Message
## Technologies Used
-> HTML
-> CSS
-> JavaScript
## Project Structure
-> index.html
-> styles.css
-> script.js
## How It Works
-> Enter your email address
-> Enter Your Password
-> Click the Login Butoon
-> Validate user input
-> Store user data using Local Storage
